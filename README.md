# BAZ Group Website

Dieses Projekt ist eine Unternehmenswebsite für die BAZ Group — eine Unternehmensgruppe im Bereich Industrie, Maschinenbau und Engineering.

## 🌐 Live Demo

[Visit Website](https://baz.by)

## 📌 Überblick

Die Website bietet eine strukturierte Übersicht über die Unternehmen der BAZ Group, einschließlich ihrer Tätigkeitsbereiche, Spezialisierungen und Kontaktdaten.  
Die Anwendung wurde als leichtgewichtige, performante statische Website umgesetzt.

## 🚀 Funktionen

- Mehrsprachige Unterstützung (RU / EN / ES / PT / ZH)
- Übersicht über alle Unternehmen der Gruppe
- Responsives Design
- Einfaches und klares UI
- Keine Backend-Abhängigkeit (statische Seite)

## 🏗️ Projektstruktur

```text
/ (root)
├── index.html   # Einstiegspunkt der Anwendung
├── styles.css   # Globale Styles
├── js/          # JavaScript-Logik
├── img/         # Bilder und Assets
└── llcs/        # Seiten einzelner Unternehmen
```
